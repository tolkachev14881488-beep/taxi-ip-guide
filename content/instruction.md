# Инструкция по открытию ИП для работы в такси (РБ)

## 1. Введение: зачем ИП для такси

Для легальной работы водителем такси через агрегаторы (Яндекс Go, Uber и др.) в Республике Беларусь необходим статус индивидуального предпринимателя или самозанятого.

**Преимущества ИП:**
- Легальное получение дохода
- Возможность подключения к агрегаторам такси
- Упрощённая система налогообложения для ИП

---

## 2. Подготовка документов

Перед регистрацией подготовьте:

- [ ] Паспорт гражданина РБ
- [ ] Идентификационный номер (ИНН)
- [ ] Адрес регистрации (прописка)
- [ ] Контактный телефон и email
- [ ] Учётная запись на портале [egovernment.by](https://egovernment.by)

> **Совет:** Заранее проверьте, что ваш ИНН активен и данные в госреестрах актуальны.

---

## 3. Регистрация ИП в МНС через egovernment.by

### Шаг 1: Авторизация
1. Перейдите на [egovernment.by](https://egovernment.by)
2. Войдите через мобильный ключ, ЭЦП или банковский ID

### Шаг 2: Подача заявления
1. Раздел «Регистрация субъектов хозяйствования»
2. Выберите «Регистрация индивидуального предпринимателя»
3. Укажите код вида деятельности (ОКЭД) — для перевозки пассажиров
4. Выберите систему налогообложения
5. Подпишите заявление ЭЦП

### Шаг 3: Получение свидетельства
- Срок рассмотрения: обычно 1–3 рабочих дня
- Свидетельство будет доступно в личном кабинете

---

## 4. Выбор системы налогообложения

Для водителей такси часто выбирают:

| Система | Особенности |
|---------|-------------|
| **УСН без НДС** | Упрощённый учёт, фиксированные ставки |
| **Единый налог** | Для отдельных видов деятельности |

> **Важно:** Проконсультируйтесь с бухгалтером или МНС для выбора оптимальной системы под ваш случай.

---

## 5. Подключение к агрегатору такси

После регистрации ИП:

1. **Яндекс Go (Pro):**
   - Скачайте приложение Яндекс Pro
   - Зарегистрируйтесь как водитель-партнёр
   - Загрузите документы: свидетельство ИП, права, техпаспорт, страховка

2. **Другие агрегаторы:**
   - Проверьте требования на сайте выбранного сервиса
   - Подготовьте те же базовые документы

### Необходимые документы для агрегатора:
- [ ] Свидетельство о регистрации ИП
- [ ] Водительское удостоверение
- [ ] Свидетельство о регистрации ТС
- [ ] Полис ОСАГО
- [ ] Лицензия такси (если требуется в вашем регионе)
